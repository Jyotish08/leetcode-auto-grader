#leetcode auto grader
an AI powered system that evaluates my leetcode solutions and tracks progress over time.

## features
- AI-based code evaluation
- static code analysis(loops and recursion)
- automatic complexity estimation
- score calculation system
- result tracking using csv
- progress analytics report
## project architecture
user code
static analysis
AI evaluation
score calculation
result storage
progress analytics
## how it works
1.  user enters problem name
2.  user paste code solution
3.  system analyzesthe code structure
4.  AI evaluate algorithm quality
5.  score is calculated using a grading schema
6.  results are stored and tracked
